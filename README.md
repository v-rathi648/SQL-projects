# Data Cleaning Project.sql
## Overview                                                                                                                                  This project focuses on cleaning and preparing raw data using SQL to make it analysis-ready . The dataset initially contained inconsistencies such as missing values, duplicates, irrelevant entries. Through a series of SQL queries, the data was transformed into a structured and reliable format.                                                                                                             
## Key Tasks Performed :                                                                                                                      - Removed duplicate records to ensure data integrity.                                                                                   - Handled missing/Null values using appropriate techniques.                                                                             - Standardized inconsistent data formats.                                                                                               - Trimmed and cleaned unwanted spaces and characters.                                                                                                  
## Objective :                                                                                                                                 To demonstrate strong data cleaning and preprocessing skills using SQL, which is crucial step before performing any data analysis or visualization.                                                                                                                              
## Tools Used :                                                                                                                                 - MYSQL                                                                                                                                                        
## Why This Project Matters :                                                                                                                  Clean data is the foundation of every great analysis.This project showcases my ability to handle real-world data challenges and build a strong base for data-driven insights-an essential skill for any Data Analyst role.                                                
## Final Note :                                                                                                                                This project is a part of my journey towards becoming a data analyst, focusing on building practical, job-ready SQL skills through hands-on learning.
